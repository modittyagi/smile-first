# 🦷 Smile First Dental Clinic Website

A professional, mobile-first dental clinic website built to industry standards.

## 📁 Project Structure

```
smile-first-dental/
├── index.html      # Main HTML file
├── styles.css      # Complete CSS stylesheet
├── script.js      # JavaScript interactions
└── README.md       # This file
```

## 🚀 Features

- **Mobile-First Design** - Fully responsive, optimized for all devices
- **WhatsApp Integration** - All booking forms send to WhatsApp
- **Click-to-Call** - Direct phone calls from mobile
- **18+ Sections**:
  - Emergency Banner
  - Sticky Header with Navigation
  - Hero Section with Trust Stats
  - Services Grid (6 services)
  - Why Choose Us
  - Doctor Profile
  - Gallery
  - Testimonials Carousel
  - FAQ Accordion
  - Location & Map
  - Contact Form
  - Footer

## 📱 Quick Start

Open `index.html` in any browser to view the website.

## 🔧 Deployment

### Option 1: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload these files
3. Go to Settings → Pages → Deploy from main branch

### Option 2: Netlify (Free)
1. Go to netlify.com and sign up
2. Drag and drop the `smile-first-dental` folder
3. Your site is live!

### Option 3: Vercel (Free)
1. Go to vercel.com and sign up
2. `npm i -g vercel`
3. Run `vercel` in the project folder

## 🛠️ Customization

### Change WhatsApp Number
Edit `script.js` line 4:
```javascript
const PHONE_NUMBER = '917666421234';
```

### Change Clinic Details
Edit `index.html`:
- Lines 40-50: Phone numbers
- Lines 330-380: Doctor info
- Line 500+: Address

### Change Colors
Edit `styles.css` CSS variables:
```css
:root {
    --primary: #0D9488;  /* Main teal color */
    --accent: #F59E0B;   /* Gold accent */
}
```
